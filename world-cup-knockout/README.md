# World Cup Knockout Predictor

A mobile-first knockout bracket app for prediction practice and code review.

## Features

```text
Full-screen dark bracket view
Smooth horizontal scrolling
Round navigation
Click a team to pick a winner
Winner automatically advances
Predicted champion display
Local browser storage
Reset button
News / Wiki / Images links per team
```

## Data Status

The starter bracket is demonstration data based on the visual pattern of a knockout bracket. It is not live FIFA data.

Update `script.js` when official fixtures, teams and match times are required.
